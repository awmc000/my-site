School

I am working on my Bachelor's of Science in Computer Science at Vancouver Island University in Nanaimo. I'm about halfway through my degree, and have taken classes in Java and C++ programming, software engineering, computer architecture, calculus and discrete math, among others.

I'm currently enrolled in a course called Systems and Networks, which involves learning to configure and maintain Linux environments for different scenarios, and other, more general CS topics, like how hard drives, memory, and computer networks work. I've come to really like Linux so it's been great. The other CS course I'm taking right now is Database Systems. So far, we have worked through ER modelling and queries in SQL, Datalog and relational algebra (esp. expression trees). Again this one is super fun, I really wanted to learn about databases.
